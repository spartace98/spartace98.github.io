    April Tag Detection for Soft Robotics Hand Manipulations
    I have been working with Professor Nancy Pollard person from the Robotics Institute to design a soft robotics hand for dexterous manipulations.

    In order to test the robotic's hand ability to grab and perform manipulations to the object, the hand must be able to detect the object.

    April Tags are unique and easily detectable with any commerical camera. Current libraries are capable of providing dimensions and position of the detected tag, with careful camera calibration.



    To test feedback between the camera and the computer, I simulated a complex racing game which can be controlled by the movements of the April Tag.



    Object Detection for General Applications
    It is useful to detect any given object for general applications.

    For objects with isolated color, without machine learning techniques to understand the texture, it is difficult to analyse the angle of the given object.

    A meaningful first step that has application, however, is to detect its position from a given image with color calibration to isolate the given object from the surroundings.



    I further tested the feedback between the real time object detection and performance of a given task.



    My project has demonstrate that while it is possible to perform a task with object detection of a general object, further research can be done to improve the efficiency of this feedback.
